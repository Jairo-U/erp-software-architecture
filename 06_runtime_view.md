# Vista de Ejecución

## Escenario: Registrar Producto

1. El Administrador ingresa los datos del producto en la aplicación web.
2. La SPA envía los datos al backend mediante una petición POST.
3. El backend valida la información.
4. El backend guarda el producto en la base de datos.
5. El sistema devuelve un mensaje de éxito.
