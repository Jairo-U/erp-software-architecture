# Alcance y Contexto del Sistema

El Sistema ERP interactúa principalmente con los usuarios internos de la organización y con un sistema contable externo.

El Administrador de Compras gestiona productos, proveedores y órdenes de compra.  
Otros usuarios pueden consultar información.  
El ERP envía información contable al sistema externo.
